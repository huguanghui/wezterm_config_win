## wezterm 配置

### 快捷键速查

#### 功能键

| 快捷键 | 功能 |
|--------|------|
| `F1` | CopyMode（vi 模式滚动/选择文本） |
| `F2` | 命令面板 |
| `F3` | 启动器 |
| `F4` | 标签页模糊搜索 |
| `F11` | 全屏切换 |

#### 复制粘贴 / 滚动 / 快速选择

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 复制到剪贴板 |
| `Ctrl+Shift+V` | 从剪贴板粘贴 |
| `Ctrl+Shift+↑` / `↓` | 单行滚动 |
| `Ctrl+Shift+PageUp` / `PageDown` | 翻页滚动 |
| `Ctrl+Shift+Space` | QuickSelect 快速选择文本对象 |

#### Tab 标签页

| 快捷键 | 功能 |
|--------|------|
| `Alt+T` | 新建 Tab |
| `Alt+Ctrl+W` | 关闭当前 Tab |
| `Alt+[` / `Alt+]` | 切换到上/下一个 Tab |
| `Alt+Ctrl+[` / `Alt+Ctrl+]` | 移动当前 Tab 位置 |
| `Alt+1~9` / `Alt+Ctrl+1~9` | 跳转到第 1~9 个 Tab |

#### Window 窗口

| 快捷键 | 功能 |
|--------|------|
| `Alt+N` | 新建窗口 |
| `Alt+F` | 搜索文本 |

#### Pane 窗格

| 快捷键 | 功能 |
|--------|------|
| `Alt+\` | 垂直分割（左右） |
| `Alt+Ctrl+\` | 水平分割（上下） |
| `Alt+Enter` | 当前 Pane 最大化/还原 |
| `Alt+Ctrl+H/J/K/L` | 切换到 左/下/上/右 Pane |
| `Alt+Ctrl+P` | Pane 选择器（数字标签互换位置） |

#### 光标移动

| 快捷键 | 功能 |
|--------|------|
| `Alt+←` / `Alt+→` | 光标跳到行首/行尾 |
| `Alt+Backspace` | 删除到行首 |

#### Leader 键（`Alt+Ctrl+Space`）

按下 Leader 后释放，再按下一键：

| 快捷键 | 功能 |
|--------|------|
| `Leader f` | 进入字体缩放模式 → `j`/`k` 减小/增大，`0` 重置，`q`/`Esc` 退出 |
| `Leader p` | 进入 Pane 尺寸调整 → `h/j/k/l` 方向调整，`q`/`Esc` 退出 |
| `Leader r` | 随机切换背景壁纸 |
| `Leader R` | 重载配置文件 |
| `Leader b` | 打开常用命令菜单 |

#### 鼠标

| 操作 | 功能 |
|------|------|
| `Ctrl+左键单击` | 打开鼠标下的链接 |
| 新标签按钮 `+` 右键 | 打开启动器菜单 |
